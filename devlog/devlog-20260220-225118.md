## Issue #31 - Remove 'Game loaded' popup

Removed the "Game loaded! Click to start playing." popup that appeared on page load. The `showMessage()` call in `js/main.js` was removed from the DOMContentLoaded handler. The `showMessage` function itself is retained as it's used elsewhere (pause notifications, debug mode). Added playtester test T2-19 to verify no popup appears on startup.

## Issue #33 - Redesign first level into multi-room nuclear reactor

Expanded the map from 16x16 to 24x24 and redesigned it as a multi-room nuclear reactor facility with five distinct interconnected areas:

- **Control Room** (wall type 4/tech) — player spawn area in the top-left, a compact room with tech-textured walls
- **Containment Wing** (wall type 2/brick) — medium rooms on the east side with brick walls
- **Reactor Core** (wall type 5/marble) — large central chamber with marble walls, heavily guarded
- **Cooling Tunnels** (wall type 3/metal) — narrow metal corridors flanking the reactor core
- **Waste Storage** (wall type 6/stone) — dead-end rooms in the south-west with bonus pickups

Enemies are distributed across all four quadrants (9 total, 8 types). Pickups placed in each area to reward exploration. Doors added at chokepoints between areas, including a red key door to the south section. A* pathfinding max iterations increased from 200 to 500 for the larger map. Added playtester test T2-20 to verify the multi-room layout.
